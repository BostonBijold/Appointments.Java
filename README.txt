Created Using: IntelliJ Community 2020.3 with Java JDK 11.0..9 and javafx-sdk-11.0.2

To run the program through a Intellij, open the program folder in the IDE and select run.

C195_Project is a consultation appointment scheduling applacation. Used to schedule 30 minute consultations between 8:00 AM and 10:00 PM EST.
Only one appointment can be scheduled for each timeslot each day. 
Once logged in the user can add, update and delete appointments and customers. Users and Contacts must be added through the database directly. 

The third report is a customer total report that shows the customer ID, customer Name and total appointments scheduled by each customer to show which customers schedule the most appointments. 

SQL driver to connect to database: mysql-connector-java-8.0.22
